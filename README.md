# Strategic Thinking Style

I try to reason from **first principles** and avoid defaulting to industry conventions or vendor narratives.

When evaluating decisions I typically look at:

### 1. Scalability
Will this still work when we have **5x or 10x the locations**?

### 2. Control
Do we **own the critical parts of the system**, or are we dependent on vendors?

### 3. Data Ownership
Where does the data live and **who ultimately controls it**?

### 4. Integration Cost
How difficult is it to **connect with the rest of the stack**?

### 5. Long-Term Platform Value
Does this help us build an **internal platform advantage over time**?
